Authentication Tokens

Authentication tokens allow scripts using lavacli and other scripts based on XML-RPC to securely access LAVA resources. You can create and use any number of tokens simultaneously. If you believe a token is compromised you can quickly remove it. Anyone using that token will no longer be able to authenticate as You in the system.
